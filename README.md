# Bike Sharing Dashboard  
**Exploring Bike Sharing Data (2011-2012) in Washington D.C.**  

Dashboard ini dibuat menggunakan **Streamlit** untuk menganalisis pola peminjaman sepeda berdasarkan berbagai faktor seperti cuaca, musim, dan tren harian. Dashboard ini mendukung **visualisasi interaktif** dengan **Seaborn** dan **Matplotlib**.

---

## Cara Menjalankan Dashboard  

### 1️⃣ Install Python  
Pastikan Anda sudah menginstal **Python** di perangkat Anda.  
Jika belum, unduh dan instal dari [python.org](https://www.python.org/).

---

### 2️⃣ Clone Repository
Jika Anda belum memiliki proyek ini di perangkat lokal, unduh dengan perintah berikut:

```bash
git https://github.com/codedreamerD/Proyek-Analisis-Data.git
```
```bash
cd bike-sharing-dashboard
```

atau dapat melakukan fork/download terhadap dataset, lalu membukanya di code editor

### 3️⃣ Buat Virtual Environtment (Opsional, tetapi Direkomendasikan)
#### Untuk macOS/Linux
```bash
python -m venv venv
```
```bash
source venv/bin/activate
```

#### Untuk Windows
```bash
python -m venv venv
```
```bash
venv/bin/activate
```

### 4️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 5️⃣ Jalankan Aplikasi
```bash
streamlit run dashboard.py
```
